XML

XML (Extensible Markup Language) es un lenguaje de marcado que define reglas para la codificación de documentos en un formato legible tanto para humanos como para máquinas. Es ampliamente utilizado para el intercambio de datos estructurados en aplicaciones web y servicios API.

Ejemplo de XML

Aquí tienes un ejemplo de cómo se ve un documento XML:

<personas>
    <persona>
        <nombre>Juan</nombre>
        <edad>30</edad>
        <ciudad>Nueva York</ciudad>
    </persona>
    <persona>
        <nombre>Ana</nombre>
        <edad>25</edad>
        <ciudad>Madrid</ciudad>
    </persona>
    <persona>
        <nombre>Pedro</nombre>
        <edad>35</edad>
        <ciudad>Barcelona</ciudad>
    </persona>
</personas>

Otro ejemplo de XML

Aquí tienes otro ejemplo de un documento XML que representa una lista de libros:

<libros>
    <libro>
        <titulo>El Padrino</titulo>
        <autor>Mario Puzo</autor>
        <año>1969</año>
        <genero>Drama</genero>
    </libro>
    <libro>
        <titulo>1984</titulo>
        <autor>George Orwell</autor>
        <año>1949</año>
        <genero>Ciencia ficción</genero>
    </libro>
</libros>
